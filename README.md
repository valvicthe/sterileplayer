# sterile v1
A very simple audio player for people who want a full eq and preamp. Based off of my old project "Quellqa" or "LarpMedia".

## features
1. full EQ - allows the modification of sound frequencies on the low end, mid end and high end.
2. preamp - make the music as loud as you want.
3. library - shows all of your music you have loaded in. **unfortunately does not save at the moment.**
4. discord RPC - show your friends what your listening to!
5. themes - simple color changing themes, with more to come!

## installation
1. visit the releases tab on this repo
2. download the version of sterile that matches your operating system, for example, if i was on **Windows** i would download `sterile-binary-windows.zip`, or if i was on macos, it would be `sterile-binary-macos.zip`
3. extract the zip
4. run the app
5. done
